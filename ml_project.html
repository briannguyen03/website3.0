Drawing with Ai

What is it?

HandPose is a machine-learning model that allows for palm detection and hand-skeleton finger tracking in the browser. It can detect multiple hands at a time and for each hand, and provides 21 2D and 3D hand keypoints that describe important locations on the palm and fingers. The ml5.js HandPose model is based on the HandPose implementation by TensorFlow.js.
- ML5.js

The following functionality is provided:

  1. Hand Keypoint Detection: HandPose can detect the 2D and 3D coordinates of 21 keypoints on a hand.
  2. Handedness: HandPose can determine the handedness (left or right) of the detected hand.
  3. Multiple Hands: HandPose can detect multiple hands at the same time.


What I learned

Building on what I learned from the handpose projects and with guidance from the Coding Train‚Äôs ML playlist, I created my own project using the HandPose model.

This project combines real-time hand tracking with logic to detect the distance between the index finger and thumb on both hands. The left hand acts as the "brush" when the user pinches their index finger and thumb together, a circle is painted on the canvas at that X/Y coordinate. On the other hand (literaly), users can control the size of the circle by adjusting the distance between the index finger and thumb on their right hand.

Since the HandPose model is based on MobileNetV1, it supports real-time tracking even on mobile devices, though performance may vary depending on the device's processing power.
